# React TODO アプリ

このプロジェクトは Vite をベースにした React 製の TODO アプリです。入力フォームからタスクを追加し、完了状態の切り替え・削除・フィルタリング・検索ができます。タスクはローカルストレージに保存され、ページを再読み込みしても保持されます。

## 主な機能

- タスクの追加／完了／削除
- すべて・未完了・完了済みの切り替えフィルタ
- キーワードによるタスク検索
- 未完了タスク数の表示
- 完了済みタスクの一括削除
- ローカルストレージによる永続化

## セットアップ

```bash
npm install
npm run dev
```

開発サーバーは `http://localhost:5173` で起動します。

## ビルド

```bash
npm run build
```
